# Breathe Timer

Приложение для практики дыхательных упражнений на чистом JS.   
Демонстрационная версия - [Breathe Timer](https://breathe-timer.web.app/) 

## Используется:

* ES6
* Web Audio API
* SCSS
* Webpack
